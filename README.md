## for developer

```
$ npm start
```
アプリケーションをスタートします。

```
$ npm run up
```
サーバーを起動します。(`npm start` と同じ)

```
$ npm run preview
```
サーバーを起動し、標準ブラウザでプレビューを開始します。(Mac OSX のみ対応)

```
$ npm run dev
```
`preview`　と `gulp watch` を同時に実行します。

```
$ npm run test
```
テストスクリプトを実行します。
